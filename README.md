# vorp_deliveryjob

## 1. Requirements

- [VORP-Core](https://github.com/VORPCORE/vorp-core-lua)

## 2. Installation

- Clone this repository
- Extract the zip
- Put vorp_DeliveryJob to your resource folder
- Add ```ensure vorp_DeliveryJob``` in your server.cfg
- You can change the language in ```en.lua```
